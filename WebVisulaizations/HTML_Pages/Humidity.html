The purpose of this project was to analyze how weather changes as you get closet to the equator. To accomplish this analysis, we first pulled data from the Open WeatherMap API to assemble a data set on over 500 cities.

After assembling the dataset, we used Matplotlib to various aspects of the weather vs. latitude. Factors we looked at include temperature, cloudiness, wind speed and humidity. This site provides the source data and visualizations created as parts of the analysis as well as explanations and descriptions of any trends and correlations witnessed.
